# Hotel Booking Cancellation Predictor

A machine learning web app that predicts whether a hotel booking will be canceled or not based on booking details.

## Overview
Hotels lose significant revenue from booking cancellations. This project uses real hotel booking data to build a prediction model that helps hotels identify which bookings are likely to be canceled, allowing them to manage their rooms and revenue more effectively.

## Dataset
- Source: Hotel Booking Demand dataset (Kaggle)
- Size: 119,390 rows and 32 columns
- Target: is_canceled (0 = Show Up, 1 = Canceled)

## Best Model — Random Forest
- Accuracy: 89.9%
- Precision: 89.4%
- Recall: 83.1%
- F1 Score: 86.1%
- ROC-AUC: 96.1%

## Tech Stack
Python, Pandas, Scikit-learn, XGBoost, Matplotlib, Seaborn, Streamlit
